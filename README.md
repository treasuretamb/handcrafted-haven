# Handcrafted Haven

## Group Project for WDD 430 Full-Stack Development

### Description
Handcrafted Haven is a web application that provides a platform for artisans and crafters to showcase and sell their unique handcrafted items. It serves as a virtual marketplace, connecting talented creators with potential customers who appreciate the beauty and quality of handmade products.

### Technologies
- **Front-End:** HTML, CSS, TypeScript, Next.js, Tailwind CSS
- **Back-End:** Node.js
- **Deployment:** Vercel
- **Project Management:** GitHub Projects (Kanban Board)
